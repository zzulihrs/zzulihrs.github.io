字典增删改查

获取字典的value值
person['name'], person.get('name')
不存在时第一种方式会报错，.get会返回null值
删除del person['name']
del person 删除字典这个变量
person.clear()清空字典，但是保留字典对象

遍历字典所有的key： for key in person.keys():
遍历字典的value：for value in person.values():
遍历字典的key和value：for key, value in person.items():
遍历字典的项/元素：for item in person.items():

序列化的两种方式

//导入json模块到该文件中
//我们在使用scrapy框架的时候，该框架会返回一个对象，我们要将对象写入到文件中，就要使用json.dumps()

import json
names = json.dumps(name_list) #将python对象变成json字符串
fp.write(names)
fp.close()

dump将对象转化为字符串的同时，指定一个文件的对象，然后把转化后的字符串写入到文件中
json.dump(name_list, fp)#相当于names = json.dumps(name_list)和fp.write(names)

反序列化：将json字符串变成一个python对象
import json
content = fp.read()
json.loads(content)

result = json.load(fp)//直接读完变成python对象
